Коли пропонувати Volpato клієнту

Volpato — це не «просто алюмінієва рамка», а готовий інструмент для салонів, студій і виробників, які хочуть додати скляні фасади та легкі вітрини без складної розробки системи з нуля.

Кухні та високі пенали
  • Верхні модулі з підсвіткою й склом.
  • Високі вітрини для сервізів та декору.
  • Поєднання з фасадами Cleaf / SAIB.
Гардеробні та нішеві рішення
  • Скляні дверцята для гардеробних секцій.
  • Фасади для ніш у коридорі чи спальні.
  • Комбінація з системами TRIAL / eTSecutive.
Вітрини салонів та шоуруми
  • Вітрини для салонів кухонь та меблів.
  • Експозиції матеріалів, посуду, декору.
  • Продумані допуски та захист скла.

Серії профілів Volpato, які ми підтримуємо

У калькуляторі MTU використовується фіксований набір перевірених профілів Volpato. Для кожної серії закладені допуски, товщина скла та сумісні вузли — щоб ви отримували коректну специфікацію з першого разу.

AV2
Класичний фасад з тонкою рамкою
  • Легка алюмінієва рамка для скла.
  • Оптимально для кухонних фасадів.
  • Float / ExtraClear, скло 4 мм.
AV4
Посилена рамка та великі формати
  • Для вищих фасадів та пеналів.
  • Більша жорсткість по периметру.
  • Сумісність з ручками-профілями.
AV8
Мінімальна видима рамка
  • Максимальний акцент на склі.
  • Добре працює з підсвіткою.
  • Рекомендована для сучасних кухонь.
AV9
Спеціальні рішення та нішеві фасади
  • Застосовується там, де потрібна особлива геометрія.
  • Може комбінуватися з іншими серіями.
  • Точні допуски вбудовані в калькулятор.

Volpato + Busel Glass + MTU: один цикл прорахунку

Важливо не тільки порахувати алюміній, але й правильно закласти скло, розкрій та сервіс. У калькуляторі вже інтегровано методику Busel Glass v3 та послуги MTU.

Скло Busel v3
Коректні розміри та типи скла
  • Float / ExtraClear, товщина 4 мм (за потреби — інші варіанти).
  • Автоматичні допуски на скло під кожну серію профілю.
  • Формати під різ та оптимізація розкрою.
Послуги MTU
Від профілю до готового фасаду
  • Розкрій профілю Volpato під замовлення.
  • Скло Busel з обробкою кромки та, за потреби, загартуванням.
  • Збірка фасадів, упаковка та комплектація по проєкту.
